# SolucionesTecnologicas-AWS-Devops-2026
Infraestructura automatizada en AWS para Soluciones Tecnológicas del Futuro. Implementa CI/CD con GitHub Actions, contenedores Docker, microservicios con Lambda y API Gateway, e IaC con CloudFormation. Optimiza el despliegue y monitoreo mediante Python (Boto3) y CloudWatch dentro de las restricciones de AWS Learner Lab.
